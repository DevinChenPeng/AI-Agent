# 概览（Overview）

AI-Agent 是基于 Electron + React + TypeScript + Vite 的桌面应用模板，集成了：

- Electron 主/预加载/渲染三进程工程化（electron-vite）
- React 18+ 与 TypeScript 5
- Ant Design 5 组件库与图标（@ant-design/icons）
- CSS 变量 + 工具类体系，统一主题、间距、阴影、圆角
- VSCode 代码片段与 className 工具函数（cn）提高开发效率
- axios 封装的 HTTP 客户端

适用于需要快速交付跨平台桌面端应用的场景。
